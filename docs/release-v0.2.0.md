# Sopaper Evidence v0.2.0

## Summary

First public packaging pass for Sopaper Evidence.

## Highlights

- established the evidence-first skill workflow
- added source-priority, evidence-schema, and claim-audit references
- added search and evaluation playbooks for prior work, benchmarks, and baselines
- added reusable templates for evidence briefs, related work matrices, claim maps, and paper outlines
- added an OpenClaw end-to-end example chain from input to outline
- packaged the repository for GitHub and marketplace publication

## Why this release matters

This release turns the project from an internal concept into a publishable skill package with a clear trust model: search real sources first, classify evidence carefully, and refuse to overstate unsupported claims.

## Notes

- this release is documentation- and workflow-focused
- no executable code is required for the core skill behavior
- repository checks currently validate structure, links, and core metadata consistency
